# Frontend - Sistema de Captura de Prendas

Este frontend está pensado para publicarse en GitHub Pages.

## Archivos
- index.html
- styles.css
- app.js

## Importante
Este frontend no controla directamente el PLC ni la cámara.
Se comunica con un backend Flask.

## Uso
1. Abrir la página web
2. Escribir la URL del backend, por ejemplo:
   http://192.168.3.100:5000
3. Guardar URL
4. Usar los botones del sistema

## GitHub Pages
1. Subir estos archivos al repositorio
2. Ir a Settings > Pages
3. Branch: main
4. Folder: /(root)
5. Guardar
